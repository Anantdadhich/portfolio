## üöÄ Getting Started

To run this project locally:

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/Anantdadhich/portfolio
    cd portfolio
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    # or
    pnpm install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    # or
    yarn dev
    # or
    pnpm dev
    ```

4.  **Open in Browser:**

    Visit `http://localhost:3000` to view the application.
